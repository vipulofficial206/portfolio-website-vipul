# Portfolio Website - vipulofficial206

A **modern, responsive developer portfolio website** built with React, showcasing my skills, projects, and contact info. It includes smooth animations and interactive UI to provide a great visitor experience.

## 🚀 Live Demo

👉 https://portfolio-website-vipul.vercel.app/

---

## 💡 Overview

This project is a personal portfolio website designed to showcase:
- Skills & technologies I work with
- Projects I’ve completed
- Contact form for getting in touch
- Backend API for storing messages

It is fully responsive and user‑friendly, making it ideal for showcasing my developer profile online.

---

## 🧰 Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Framer Motion  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Deployment:** Vercel (frontend), Render (backend)

---

✔️ Hero section with animated intro

## 📥 Getting Started

### Clone the repository

```bash
git clone https://github.com/vipulofficial206/portfolio-website-vipul.git
cd portfolio-website-vipul
```

🛠 Install Dependencies

Frontend
```bash
cd client
npm install
```

Backend
```bash
cd server
npm install
```

⚙️ Environment Variables

Create a .env file in both frontend and backend with values like:

Frontend
```env
VITE_API_URL=https://your-backend-url.com
```

Backend
```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
ADMIN_PASSKEY=your_secret_key
```


📦 Run Locally

# Frontend
npm run dev

# Backend
npm run start

Open http://localhost:3000 in your browser after running both servers.

📐 Deployment

The frontend is deployed on Vercel

The backend API is deployed on Render

Ensure CORS is configured to allow your frontend origin
